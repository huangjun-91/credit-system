# 教师获奖证书管理系统 🏆

一个专为高校教师设计的获奖证书登记管理系统，支持上传指导学生获奖证书图片、选择获奖层次与等级、折算学分、管理员审核统计。

## 功能

### 教师端
- 🏅 **获奖登记**：登记指导学生获奖信息，填写获奖名称、层次、等级
- 🖼️ **证书上传**：支持 JPG/PNG/PDF 格式，16MB 以内
- 📊 **层次统计**：按国家/省/市/校/院级自动汇总折算学分
- 👀 **审核进度**：实时查看每条申请的审核状态

### 管理员端
- ✅ **一键审核**：通过/拒绝，可填写审核意见
- 👥 **教师管理**：查看每位教师的获奖详情
- 📥 **数据导出**：一键导出 CSV 统计文件（含获奖层次、等级、折算学分）

## 快速启动

### 方法 1：直接运行（需要 Python）

```bash
# 1. 安装 Python（如果没有的话）
# 下载：https://www.python.org/downloads/
# 安装时勾选 "Add Python to PATH"

# 2. 安装依赖
pip install flask

# 3. 启动
python app.py
```

浏览器访问：http://127.0.0.1:5000

## 默认账号

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 管理员 | admin | admin123 |
| 教师 | 注册获取 | 自行设置 |

### 注册字段
- 用户名、密码、真实姓名
- 工号、职称（教授/副教授/讲师等）
- 所属学院/部门

### 获奖登记字段
- 获奖名称/赛事名称
- 获奖层次：国家级、省部级、市厅级、校级、院级
- 获奖等级：一等奖、二等奖、三等奖、特等奖、优秀奖、入围奖、其他
- 折算学分数
- 获奖说明/指导学生信息
- 证书图片上传

## 项目结构

```
credit-system/
├── app.py              # 主程序
├── requirements.txt    # 依赖
├── templates/          # 前端模板
```

## 技术栈

- **后端**：Python Flask
- **前端**：Bootstrap 5 + Bootstrap Icons
- **数据库**：SQLite（零配置）
- **图片存储**：本地文件系统
